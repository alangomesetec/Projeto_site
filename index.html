HTML & CSS

O que é CSS?

  • CSS significa Cascading Style Sheets;
  • É uma "folha de estilo" composta por “camadas”;
  • Utilizada para definir a apresentação (aparência);
  • É uma extensão de linguagens de marcação como XML, HTML e XHTML.

Tipos de CSS

  1. Inline - usando o atributo style dentro dos elementos HTML;
  2. Interno - usando um elemento style na seção head;
  3. Externo - usando um elemento link para vincular a um arquivo CSS externo.

O que é HTML?

  • HTML significa HiperText Markup Language;
  • HTML é uma linguagem de marcação utilizada na construção de páginas na Web;
  • HTML consiste em uma série de elementos.
